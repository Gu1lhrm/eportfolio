ePortfolio

Análise e recomendações de otimização para motores de busca, alinhadas à comunicação centrada no utilizador do WRIT 2201.

The ePortfolio project is the final project for the WRIT2201: Professional Writing course, where I must present the work I've done during the semester (definition, steps, personal contributions), reflections, and how I can apply it to future work. The development process involves taking information from past work, reflecting on what was done, developing the website, and presenting all the points mentioned previously in a clean, organized, and intuitive way.

I start by taking information from previous personal progress reports to familiarize myself with what was done, what I learned during the process, and how I can apply what I learned to future work. With this, I created a draft with the general information that will be presented in my ePortfolio. With the information already organized, I started developing the website using the HTML and CSS programming languages ​​through the Visual Studio Code IDE. With the website completed, I improved my general information using peer feedback and incorporated it into the final website.

Imagens do projeto

My contributions

In this project, I was responsible for writing, designing, organizing, and developing the entire ePortfolio. I started by writing the content of the ePortfolio, which involved taking information from previous progress reports and organizing it in a way that would be clear and engaging for readers. After writing the content, I focused on designing the layout and visual elements of the ePortfolio to ensure that it was visually appealing and easy to navigate. Finally, I developed the website using HTML and CSS, implementing the design and ensuring that all elements were functional and responsive across different devices.

Reflections

I believe the most important thing to take away from this project is the opportunity to look at past projects and have the ability to objectively analyze what was done, what I learned, how I can improve for the future, and how I can apply these learnings to future projects - both in the academic and professional world. Since I decided to program the ePortfolio from scratch using HTML and CSS on the final website, my prior knowledge of these languages ​​contributed greatly to the success of my project. At the same time, it was what helped me the most, and it was also the thing I found most interesting about the process. It had been years since I had done a project using these languages, and it was a good refresher of this knowledge. Strengths I learned about myself were my ability to work with deadlines and stick to a plan.

Application

O processo de desenvolvimento deste site me ensinou maneiras de apresentar informações técnicas mais complexas - como explicar o que é o Fast Fix guid - de uma forma que seja acessível para pessoas que não têm o conhecimento prévio do projeto, como pessoas que não fizeram o curso ou possíveis empregadores.
